Erhobene Daten

  • Sitzungs-ID: Eine zufällig generierte Session-ID wird serverseitig gespeichert, um den Gesprächsverlauf innerhalb einer Sitzung zu ermöglichen. Sie enthält keine personenbezogenen Daten.
  • Fragen & Antworten: Eingegebene Fragen und die zugehörigen Antworten werden zur Qualitätssicherung und Weiterentwicklung gespeichert. Bitte gib keine persönlichen Daten ein.
  • Feedback: Bewertungen (👍/👎) werden der jeweiligen Frage zugeordnet gespeichert.
  • Zugangscodes: Eingelöste Codes werden zur Nutzungskontrolle gespeichert. Persistent-Codes werden zusätzlich im LocalStorage des Browsers abgelegt.
  • Theme-Einstellung: Die gewählte Darstellung (hell/dunkel) wird im LocalStorage gespeichert.
  • Filtereinstellungen: Wenn „Filter merken" aktiviert ist, werden die gewählten Filteroptionen (Gremium, Zeitraum, Antwortlänge, Qualität, Quellenanzahl) im LocalStorage des Browsers gespeichert.

Keine Tracking- oder Analyse-Tools

Es werden keine Cookies zu Werbezwecken gesetzt. Es kommen keine externen Analyse- oder Tracking-Dienste zum Einsatz.

Externe Dienste

Zur Beantwortung der Fragen wird eine API auf einem separaten Server ([Backend-Domain]) angefragt, die ein KI-Sprachmodell nutzt. Die eingegebene Frage wird an diesen Dienst übermittelt.

Löschung

Das Zurücksetzen des Gesprächs (↺) löscht den Sitzungsverlauf. Der LocalStorage kann über die Browser-Einstellungen gelöscht werden.
